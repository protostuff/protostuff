Documentation
=============

http://code.google.com/p/protostuff/

Requirements
============

Java 1.7 or higher

Build Requirements
==================

Maven 3.2.3 or higher

Building
========

```
$ mvn install
```

Build Output
============

- the generated jars are located on the "dist" dir
- the javadocs are located on the "javadoc" dir
